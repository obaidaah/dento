Dashboard

1- Date Range

  • The Date range is "Today" by default
  • You can Switch the date rang and select the Date Range you wish as follows;
  • The "Right Arrow" & the "Left Arrow" allows you to seamlessly switch the days.
  • Click anywhere on this box to select the Date Range you wish.

2- Appointment's Doctor

  • If you wish to Tailor the Dashboard for specific Doctor / Dentist / Operator, to show only all the "Appointments, Invoices, Payments"
  • concerning this Doctor, Dentist, or Operator, follow these steps;
  • Click on the "Doctor" box.
  • Select the questionable Doctor.

3- Invoice Assigned To

  • If you wish to only show the Invoices assigned to a particular user (The user might be the Secretary, Receptionist, Accountant, etc.), follow these steps
  • Click on the "Assigned to" box.
  • Choose the User in question

4- Appointment's Section

  • in this section all the appointments will appers of the selected date and doctor 
  • if the Doctor appointments box left empty this secion will show all the appointments of the day
  • every appointments will appers with differint color accourding to it it is statuts ( open - confermid - completed ....etc)
  • if you want to change the appointment statuts you can press on the pencil beside each appointment and change from this form

5- Invoices's Section

  • all invoices of the Patients of the selected date will apper in this section
  • if you press on ( add payment )  new form popup to fill details about the payment if it's (cash - card - insurance ) and the amount of cash
  • after adding the cash to any invoice - the payments will show up on the payments section
  • if you want to show the details of this invoice press on ( + ) to show up all operations which require to pay in this invoice
  • if you want to print the invoice press on the ( print icon ) on the invoice 
  • or you can send it to the patient as a link through sms or whatsapp if you press on QR icon beside print icon

6- Payment's Section

  • in this section all Payments of the Patients will show up here with the type of payment 
  • you can print the payment if you pressed on ( print ) icon
  • or you can send it to the patient as a link through sms or whatsapp if you press on QR icon beside print icon
  • Optional : the patient can sign on payment if you press on ( sign ) button
